Amazon Bedrock AgentCore について

AgentCore は、AIエージェントの開発・デプロイ・管理を加速するためのサービス群です。

## 主要コンポーネント

### AgentCore Runtime
サーバーレスのエージェント実行環境。Strands Agents、LangChain、CrewAI などのフレームワークに対応。
コンテナまたはコードベースでデプロイ可能で、自動スケーリングを提供します。

### AgentCore Gateway
既存のAPIやLambda関数をMCPツールとしてエージェントに公開するゲートウェイ。
Cognito OAuth による認証、セマンティック検索によるツール発見をサポートします。

### Managed Harness
コードを書かずにエージェントを構成できるマネージド環境。
モデル、システムプロンプト、ツールの3要素を設定するだけで動作します。

### AgentCore Memory
会話履歴やユーザー情報を永続化するメモリサービス。
短期記憶（セッション内）と長期記憶（セッション跨ぎ）をサポートします。

## Gateway Kit の活用シナリオ

Gateway Kit を使うと、以下のようなシナリオを実現できます：

1. ナレッジベース検索: 社内ドキュメントをS3にアップロードし、自然言語で検索
2. Web情報取得: 外部Webページの内容を取得して要約
3. AWSドキュメント検索: AWS公式ドキュメントから技術情報を検索

これらのツールはすべて1つのGateway経由で利用でき、
Managed Harnessから接続するだけでエージェントが活用可能になります。
